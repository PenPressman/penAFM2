penAFM2 is a Python toolkit for batch-processing Asylum Research AFM data in Igor Binary Wave (`.ibw`) format: it parses multi-channel scans with `igor2`, removes tilt via least-squares plane subtraction, computes surface roughness parameters (Ra, Rq, Rz), and exports a publication-quality `afmhot` height map as a 300 dpi TIFF with a calibrated scale bar and nm colorbar — all in one pipeline driven by `python -m src.cli --dir /path/to/data`, which writes a `summary.csv` alongside the TIFFs and skips any files that fail to parse without aborting the run. To install dependencies, run `pip install -r requirements.txt`. To run the test suite, run `pytest` from the repo root. Note that `.ibw` fixture files are excluded from the repository (see `.gitignore`) and must be supplied separately; place them in `tests/fixtures/` before running the tests.
